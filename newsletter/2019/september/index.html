Safari 13: what's new

Safari 13 brings many new privacy features that break incognito detection and old methods of fingerprinting.

Apple announced that the current Safari version now has Enhanced Anti-Fingerprinting Protection. Old methods of incognito detection were also removed.

We are pleased to announce that FPJS PRO has been fully updated to fix both of these issues. PRO version offers more accurate identification and incognito mode detection than any of our competitors on both the desktop and mobile versions of the new Safari.
